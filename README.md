# SNU MBA Course Review Guide - Unified Final

This version uses one unified review dataset.

- Korean and English reviews are counted together.
- Korean and English screens show the same course content; only the display language changes.
- Original full review text and past exam content are not included.
- Search works with Korean and English course/professor names where aliases are available.

Run:

```cmd
python -m pip install -r requirements.txt
python -m streamlit run app.py
```
